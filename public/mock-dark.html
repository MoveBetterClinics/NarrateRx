// the architecture

Six things no one else does at once.

Pieces of NarrateRx exist in other tools — the transcription, the LLM, the publishing. The leverage is in the combination.

01 / source

Interview-first architecture

The only tool in this category that starts with your voice in audio form. Castmagic starts with a podcast. Jasper starts with a prompt. We start with you talking.

02 / fanout

Four channels, one interview

Blog, social, email, and GBP drafts in parallel from a single transcript. Single-channel AI tools require four interviews and four briefs.

03 / loop

Voice fingerprint tuning

Accept and reject signals tune next month's drafts. Generic LLMs have no memory; we have a feedback loop that compounds.

04 / depth

Vertical-specific prompts

Chiro, equine, animal chiro, and coaching all get tuned audience descriptions and link libraries. Horizontal AI writers don't.

05 / ship

Native publish, not export

Buffer for social + GBP. WordPress and Astro for the blog. One click, not a copy-paste tour through five tabs.

06 / fit

Built by an operator

Every workflow decision pressure-tested in a real three-clinic practice. We optimize for "30 minutes a week between patients."
